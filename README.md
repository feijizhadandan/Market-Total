前端 Vue 项目
实现个人电商网站 GuMarket

- src
  - components: 通用组件
  - router: 路由配置
  - store: vuex 信息
  - views: 界面
  - App.vue: 根组件

后端 SpringBoot 项目

- admin: 业务模块
  - com.zhen.admin: 所有业务文件
- common: 共用工具模块
- framework: 整合框架模块
  - com.zhen.framework
    - config: 整合框架的配置文件
    - security: SpringSecurity 授权认证的实现文件
    - email: JavaMail 配置文件
    - utils: 工具类
